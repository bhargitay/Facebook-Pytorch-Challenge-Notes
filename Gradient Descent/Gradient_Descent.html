I. Gradient Descent¶

Definition¶

  • Graident Descent is an optimization algorithm.
  • Its goal is to minimize the error function
    • We can think of it as walking down from a mountain trying to reach the ground in the shortest possible way (find the the lowest error value).

How does it work?¶

We start by defining random initial parameters and from there the Gradient Descent iteratively adjusts them (using calculus) so that they minimize the given loss function.

Why do we need Gradient Descent?¶

By minimizing the loss with respect to the network parameters, we can find configurations where the loss is at a minimum and the network is able to predict the correct labels with high accuracy.

    STEP 2 - Define the error (loss) function¶

    For every point, we calculate the error.

    The error is

    • high for the misclassified points and
    • small fo correctly classified points

    Log-loss error function¶

    $$Error(y, \hat{y}) = - y \log(\hat{y}) - (1-y) \log(1-\hat{y})$$

    Log-loss error function:
  • classification loss function
  • measures the performance of a classification model where the prediction input is a probability value between 0 and 1
  • quantifies the accuracy of a classifier by penalising false classifications
  • minimising the log-loss is basically equivalent to maximising the accuracy of the classifier
    • We will use Gradient Descent to minimize this error function.
    • We are going to try to jiggle the line around to see how we can decrease the error function.
      • This will give rise to new weights W' and b' which will give us better predictions: $$\sigma(W'x + b') $$

    STEP 3 - The Gradient Descent Step¶

    Learning rate:¶

    Learning rate:
  • it determines how big steps the Gardient Descent takes into the direction of the local minimum (i.e. how fast or slow we will move towards the optimal weights)
  • we have to set the it to an appropriate value, which is neither too low nor too high

    • 3.1. Update weights¶

    For every point (coordinates) $x_1,...,x_n$ we update $ w_i $ by adding the learning rate $\alpha$ times the partial derivative of the error function respect to $ w_i $:

    $$ w_i \longrightarrow w_i + \alpha (y - \hat{y}) x_i$$

    $$ b \longrightarrow b + \alpha (y - \hat{y})$$

    In [11]:
    # Gradient descent step
def update_weights(x, y, weights, bias, learnrate):
    output = output_formula(x,weights,bias)
    wi = weights + ((learnrate*(y - output))*x)
    b = bias + (learnrate*(y-output))
    return wi, b

epochs = 100
learnrate = 0.01

def train(features, targets, epochs, learnrate, graph_lines=False):
    
    # 1. CALCULATING THE GRADIENT DESCENT  
    
    errors = []
    n_records, n_features = features.shape
    last_loss = None
    weights = np.random.normal(scale=1 / n_features**.5, size=n_features)
    bias = 0
    for e in range(epochs):
        del_w = np.zeros(weights.shape)
        for x, y in zip(features, targets):
            output = output_formula(x, weights, bias)
            error = error_formula(y, output)
            weights, bias = update_weights(x, y, weights, bias, learnrate)
            
    # 2. CALCULATING LOG-LOSS
    
        # Printing out the log-loss error on the training set
        out = output_formula(features, weights, bias)
        loss = np.mean(error_formula(targets, out))
        errors.append(loss)
        if e % (epochs / 10) == 0:  # 10 updates with the current training loss and accuracy
            print("\n========== Epoch", e,"==========")
            if last_loss and last_loss < loss:
                print("Train loss: ", loss, "  WARNING - Loss Increasing")
            else:
                print("Train loss: ", loss)
            last_loss = loss
            predictions = out > 0.5
            accuracy = np.mean(predictions == targets)
            print("Accuracy: ", accuracy)
        if graph_lines and e % (epochs / 100) == 0:
            display(-weights[0]/weights[1], -bias/weights[1])
            
    # 3. PLOTTING THE RESULTS
    
    # Plotting the solution boundary
    plt.title("Solution boundary")
    display(-weights[0]/weights[1], -bias/weights[1], 'black')

    # Plotting the data
    plot_points(features, targets)
    plt.show()

    # Plotting the error
    plt.title("Error Plot")
    plt.xlabel('Number of epochs')
    plt.ylabel('Error')
    plt.plot(errors)
    plt.show()
